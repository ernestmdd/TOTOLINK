# Overview

Critical unauthorized command execution has been found in the TOTOLINK A7000R router. The vulnerability can be triggered via route /cgi-bin/cstecgi.cgi. An attacker can achieve unauthorized RCE by sending an HTTP POST request, specifically by making a request to setting/setUpgradeUboot.
